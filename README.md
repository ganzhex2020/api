# movie

#### 介绍
SpringBoot+SpringSecurity+MyBatis-Plus 框架的前后端电影项目，前端使用vue+element ui

#### 软件架构
前后端分离项目


#### 安装教程

- 1.  api为后端项目，找到 api/src/main/resources/sql/table.sql 在数据库中执行
- 2.  修改application.yaml 配置文件
- 3.  三个vue项目首先 npm install 然后分别运行
